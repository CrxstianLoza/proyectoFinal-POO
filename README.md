# Proyecto Final - Figuras Planas

Aplicación web interactiva para trabajar la lección "Aprendamos sobre figuras planas".

## Tecnologías
- HTML
- CSS
- JavaScript
- Node.js
- Express
- TypeScript
- MySQL

## Configuración

Crear archivo .env con:

DB_HOST=127.0.0.1
DB_USER=root
DB_PASSWORD=tu_password
DB_NAME=nombre_de_tu_base
DB_PORT=3306
PORT=3000
