# Proyecto Final - Figuras Planas

Aplicación web interactiva para trabajar la lección "Aprendamos sobre figuras planas".

## Tecnologías
- HTML
- CSS
- JavaScript
- Node.js
- Express
- TypeScript
- MySQL

## Configuración

Crear archivo .env con:

DB_HOST=localhost
DB_USER=root
DB_PASSWORD=backopp9
DB_NAME=db_appfirst
DB_PORT=3306

## Base de datos

Antes de ejecutar el proyecto, abrir MySQL Workbench y correr el archivo:

database.sql

Este archivo crea la base de datos, las tablas principales y las preguntas iniciales del id 1 al 23.
